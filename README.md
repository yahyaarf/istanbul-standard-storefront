# Istanbul Standard — Static Storefront

A responsive, no-build-dependency storefront for premium Turkish menswear in Morocco.

## Open it

1. Download and unzip the folder.
2. Double-click `index.html`.
3. It will run in any modern browser.

For the best local development experience, run a simple local server from the project folder:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Included

- Responsive premium black storefront
- Six product cards at **699 DH**
- Size picker and quick product view
- Working shopping bag with browser persistence
- Search modal and collection filters
- Size guide, email-capture interaction, mobile navigation
- Placeholder product visuals in `/assets`

## Before launch

Replace the placeholder SVG product images with your real photography, connect a Moroccan payment provider, add an order/stock backend, and point the Instagram / email / phone links to your real business accounts.
